# Proxy Manager Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Dockerized Laravel 12 and Vue 3 Proxy Manager with CRUD, queued proxy checks, scheduler-driven refresh, status history, and tests.

**Architecture:** The project is a modular Laravel monolith with a Vue 3 TypeScript SPA. Backend HTTP controllers remain thin and delegate to Form Requests, Resources, Actions, queue Jobs, and a `ProxyCheckerInterface` implementation. Docker Compose runs nginx, php-fpm, MySQL, Vite, queue worker, and scheduler as separate services.

**Tech Stack:** PHP 8.3 runtime for Laravel 12, Vue 3, TypeScript, Vite, Vitest, MySQL 8, Docker Compose, Laravel Queue database driver, Laravel Scheduler, Laravel HTTP Client.

---

## Scope Check

The spec covers one cohesive MVP: proxy CRUD, queued checks, scheduler refresh, and the Vue management UI. It has multiple layers, but the layers must ship together for the product to be usable, so this plan keeps them in one implementation path and splits work into independently testable tasks.

```php
<?php

namespace App\Models;

use App\Enums\ProxyScheme;
use App\Enums\ProxyStatus;
use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Database\Eloquent\Model;
use Illuminate\Database\Eloquent\Relations\HasMany;

class ProxyServer extends Model
{
    use HasFactory;

    protected $fillable = [
        'name',
        'scheme',
        'host',
        'port',
        'username',
        'password',
        'identity_hash',
        'status',
        'checking_started_at',
        'last_checked_at',
        'last_success_at',
        'response_time_ms',
        'failure_reason',
    ];

    protected $hidden = [
        'password',
        'identity_hash',
    ];

    protected function casts(): array
    {
        return [
            'scheme' => ProxyScheme::class,
            'status' => ProxyStatus::class,
            'password' => 'encrypted',
            'checking_started_at' => 'immutable_datetime',
            'last_checked_at' => 'immutable_datetime',
            'last_success_at' => 'immutable_datetime',
        ];
    }

    public function checks(): HasMany
    {
        return $this->hasMany(ProxyCheck::class);
    }

    public function hasCredentials(): bool
    {
        return filled($this->username) || filled($this->password);
    }

    public function displayAddress(): string
    {
        $host = filter_var($this->host, FILTER_VALIDATE_IP, FILTER_FLAG_IPV6)
            ? '['.$this->host.']'
            : $this->host;

        $credentials = filled($this->username) ? rawurlencode((string) $this->username).'@' : '';

        return $this->scheme->value.'://'.$credentials.$host.':'.$this->port;
    }

    public static function identityHashFor(string|ProxyScheme $scheme, string $host, int $port, ?string $username): string
    {
        $schemeValue = $scheme instanceof ProxyScheme ? $scheme->value : $scheme;

        return hash('sha256', implode('|', [
            strtolower($schemeValue),
            strtolower($host),
            (string) $port,
            strtolower($username ?? ''),
        ]));
    }
}
```

- [ ] **Step 8: Create host validation rule**

Create `app/Rules/ProxyHostRule.php`:

```php
<?php

namespace App\Rules;

use Closure;
use Illuminate\Contracts\Validation\ValidationRule;

class ProxyHostRule implements ValidationRule
{
    public function validate(string $attribute, mixed $value, Closure $fail): void
    {
        if (! is_string($value) || trim($value) !== $value || $value === '') {
            $fail('The proxy host must be a non-empty host without surrounding spaces.');
            return;
        }

        foreach (['://', '/', '?', '#', '@', '[', ']'] as $forbidden) {
            if (str_contains($value, $forbidden)) {
                $fail('The proxy host must not contain protocol, path, query, credentials, or brackets.');
                return;
            }
        }

        if (filter_var($value, FILTER_VALIDATE_IP)) {
            return;
        }

        if (filter_var($value, FILTER_VALIDATE_DOMAIN, FILTER_FLAG_HOSTNAME)) {
            return;
        }

        $fail('The proxy host must be a valid IPv4 address, IPv6 address, or domain.');
    }
}
```

```php
<?php

namespace App\Services\ProxyChecker;

use App\Enums\ProxyScheme;
use App\Models\ProxyServer;

class ProxyUriFactory
{
    public function make(ProxyServer $proxy): string
    {
        $scheme = match ($proxy->scheme) {
            ProxyScheme::Http => 'http',
            ProxyScheme::Https => 'https',
            ProxyScheme::Socks4 => 'socks4',
            ProxyScheme::Socks5 => 'socks5h',
        };

        $host = filter_var($proxy->host, FILTER_VALIDATE_IP, FILTER_FLAG_IPV6)
            ? '['.$proxy->host.']'
            : $proxy->host;

        $credentials = '';

        if (filled($proxy->username)) {
            $credentials = rawurlencode((string) $proxy->username);

            if (filled($proxy->password)) {
                $credentials .= ':'.rawurlencode((string) $proxy->password);
            }

            $credentials .= '@';
        }

        return "{$scheme}://{$credentials}{$host}:{$proxy->port}";
    }
}
```

```php
<?php

namespace App\Services\ProxyChecker;

use App\Application\Proxies\Data\ProxyCheckResult;
use App\Enums\ProxyCheckErrorCode;
use App\Enums\ProxyStatus;
use App\Models\ProxyServer;
use Carbon\CarbonImmutable;
use Illuminate\Http\Client\ConnectionException;
use Illuminate\Support\Facades\Http;
use Throwable;

class LaravelHttpProxyChecker implements ProxyCheckerInterface
{
    public function __construct(private readonly ProxyUriFactory $uriFactory)
    {
    }

    public function check(ProxyServer $proxy): ProxyCheckResult
    {
        $startedAt = CarbonImmutable::now();
        $started = hrtime(true);

        try {
            $response = Http::connectTimeout((int) config('proxy-manager.check.connect_timeout_seconds'))
                ->timeout((int) config('proxy-manager.check.timeout_seconds'))
                ->withOptions([
                    'proxy' => $this->uriFactory->make($proxy),
                    'allow_redirects' => false,
                ])
                ->get((string) config('proxy-manager.check.url'));

            $finishedAt = CarbonImmutable::now();
            $responseTimeMs = (int) round((hrtime(true) - $started) / 1_000_000);
            $httpStatus = $response->status();
            $successCodes = config('proxy-manager.check.success_status_codes', [200, 204, 301, 302]);

            if (in_array($httpStatus, $successCodes, true)) {
                return new ProxyCheckResult(
                    ProxyStatus::Online,
                    $startedAt,
                    $finishedAt,
                    $responseTimeMs,
                    $httpStatus,
                    null,
                    null,
                );
            }

            return new ProxyCheckResult(
                ProxyStatus::Offline,
                $startedAt,
                $finishedAt,
                $responseTimeMs,
                $httpStatus,
                ProxyCheckErrorCode::BadStatus,
                "Check URL returned HTTP {$httpStatus}.",
            );
        } catch (ConnectionException $exception) {
            return $this->failure($startedAt, $started, $this->classifyConnectionError($exception), $exception->getMessage());
        } catch (Throwable $exception) {
            return $this->failure($startedAt, $started, ProxyCheckErrorCode::UnexpectedError, $exception->getMessage());
        }
    }

    private function failure(CarbonImmutable $startedAt, int $started, ProxyCheckErrorCode $code, string $message): ProxyCheckResult
    {
        return new ProxyCheckResult(
            ProxyStatus::Offline,
            $startedAt,
            CarbonImmutable::now(),
            (int) round((hrtime(true) - $started) / 1_000_000),
            null,
            $code,
            $this->sanitize($message),
        );
    }

    private function classifyConnectionError(ConnectionException $exception): ProxyCheckErrorCode
    {
        $message = strtolower($exception->getMessage());

        return match (true) {
            str_contains($message, 'timed out') || str_contains($message, 'timeout') => ProxyCheckErrorCode::Timeout,
            str_contains($message, '407') || str_contains($message, 'proxy authentication') => ProxyCheckErrorCode::ProxyAuthFailed,
            str_contains($message, 'ssl') || str_contains($message, 'certificate') => ProxyCheckErrorCode::SslError,
            str_contains($message, 'could not resolve') || str_contains($message, 'name lookup') => ProxyCheckErrorCode::DnsError,
            default => ProxyCheckErrorCode::ConnectionFailed,
        };
    }

    private function sanitize(string $message): string
    {
        $message = preg_replace('/:\/\/[^@\s]+@/', '://***@', $message) ?? $message;

        return mb_strimwidth($message, 0, 500, '');
    }
}
```

```php
namespace App\Actions\Proxies;

use App\Application\Proxies\Data\ProxyCheckResult;
use App\Enums\ProxyCheckSource;
use App\Enums\ProxyStatus;
use App\Models\ProxyServer;
use Illuminate\Support\Facades\DB;

class ApplyProxyCheckResultAction
{
    public function execute(ProxyServer $proxy, ProxyCheckResult $result, ProxyCheckSource $source): void
    {
        DB::transaction(function () use ($proxy, $result, $source): void {
            $proxy = ProxyServer::query()->lockForUpdate()->findOrFail($proxy->id);

            $proxy->status = $result->status;
            $proxy->last_checked_at = $result->finishedAt;
            $proxy->response_time_ms = $result->responseTimeMs;
            $proxy->checking_started_at = null;

            if ($result->status === ProxyStatus::Online) {
                $proxy->last_success_at = $result->finishedAt;
                $proxy->failure_reason = null;
            } else {
                $proxy->failure_reason = $this->sanitize($result->errorMessage ?? $result->errorCode?->value ?? 'Proxy check failed.');
            }

            $proxy->save();

            $proxy->checks()->create([
                'source' => $source,
                'status' => $result->status,
                'started_at' => $result->startedAt,
                'finished_at' => $result->finishedAt,
                'response_time_ms' => $result->responseTimeMs,
                'http_status' => $result->httpStatus,
                'error_code' => $result->errorCode,
                'error_message' => $result->errorMessage ? $this->sanitize($result->errorMessage) : null,
            ]);
        });
    }

    private function sanitize(string $message): string
    {
        $message = preg_replace('/:\/\/[^@\s]+@/', '://***@', $message) ?? $message;

        return mb_strimwidth($message, 0, 500, '');
```

```php
namespace App\Actions\Proxies;

use App\Enums\ProxyCheckSource;
use App\Enums\ProxyStatus;
use App\Exceptions\DuplicateProxyException;
use App\Models\ProxyServer;
use Illuminate\Database\UniqueConstraintViolationException;

class UpdateProxyAction
{
    public function __construct(private readonly ScheduleProxyCheckAction $scheduleProxyCheck)
    {
    }

    public function execute(ProxyServer $proxy, array $data): ProxyServer
    {
        $sensitiveKeys = ['scheme', 'host', 'port', 'username', 'password'];
        $shouldCheck = collect($sensitiveKeys)->contains(fn (string $key): bool => array_key_exists($key, $data));

        if (! array_key_exists('password', $data)) {
            unset($data['password']);
        }

        $nextScheme = $data['scheme'] ?? $proxy->scheme->value;
        $nextHost = $data['host'] ?? $proxy->host;
        $nextPort = (int) ($data['port'] ?? $proxy->port);
        $nextUsername = $data['username'] ?? $proxy->username;

        $data['identity_hash'] = ProxyServer::identityHashFor($nextScheme, $nextHost, $nextPort, $nextUsername);

        if ($shouldCheck) {
            $data['status'] = ProxyStatus::Unknown;
            $data['last_checked_at'] = null;
            $data['response_time_ms'] = null;
            $data['failure_reason'] = null;
        }

        try {
            $proxy->fill($data)->save();
        } catch (UniqueConstraintViolationException) {
            throw new DuplicateProxyException();
        }

        if ($shouldCheck) {
            $this->scheduleProxyCheck->execute($proxy->refresh(), ProxyCheckSource::Manual);
        }

        return $proxy->refresh();
    }
}
```

```php
use App\Actions\Proxies\ApplyProxyCheckResultAction;

class CheckProxyStatusJob implements ShouldQueue, ShouldBeUnique
{
    use Dispatchable;
    use InteractsWithQueue;
    use Queueable;
    use SerializesModels;

    public int $tries = 1;
    public int $timeout = 30;
    public int $uniqueFor;

    public function __construct(public readonly int $proxyId, public readonly ProxyCheckSource $source = ProxyCheckSource::Auto)
    {
        $this->uniqueFor = (int) config('proxy-manager.check.unique_for_seconds');
        $this->onQueue((string) config('proxy-manager.check.queue'));
    }

    public function uniqueId(): string
    {
        return 'proxy:'.$this->proxyId;
    }

    public function handle(ProxyCheckerInterface $checker, ApplyProxyCheckResultAction $applyResult): void
    {
        $proxy = ProxyServer::query()->find($this->proxyId);

        if (! $proxy) {
            return;
        }

        $proxy->forceFill([
            'status' => ProxyStatus::Checking,
            'checking_started_at' => CarbonImmutable::now(),
        ])->save();

        $result = $checker->check($proxy->refresh());

        $applyResult->execute($proxy, $result, $this->source);
    }

    public function failed(Throwable $exception): void
    {
        $proxy = ProxyServer::query()->find($this->proxyId);

        if (! $proxy) {
            return;
        }

        app(ApplyProxyCheckResultAction::class)->execute(
            $proxy,
            new ProxyCheckResult(
                ProxyStatus::Offline,
                CarbonImmutable::now(),
                CarbonImmutable::now(),
                null,
                null,
                ProxyCheckErrorCode::UnexpectedError,
                mb_strimwidth($exception->getMessage(), 0, 500, ''),
            ),
            $this->source,
        );
    }
}
```

```php
<?php

namespace App\Jobs;

use App\Actions\Proxies\ApplyProxyCheckResultAction;
use App\Actions\Proxies\ScheduleProxyCheckAction;
use App\Application\Proxies\Data\ProxyCheckResult;
use App\Enums\ProxyCheckErrorCode;
use App\Enums\ProxyCheckSource;
use App\Enums\ProxyStatus;
use App\Models\ProxyServer;
use Carbon\CarbonImmutable;
use Illuminate\Bus\Queueable;
use Illuminate\Contracts\Queue\ShouldQueue;
use Illuminate\Foundation\Bus\Dispatchable;
use Illuminate\Queue\InteractsWithQueue;
use Illuminate\Queue\SerializesModels;

class DispatchDueProxyChecksJob implements ShouldQueue
{
    use Dispatchable;
    use InteractsWithQueue;
    use Queueable;
    use SerializesModels;

    public function handle(ScheduleProxyCheckAction $scheduleProxyCheck, ApplyProxyCheckResultAction $applyResult): void
    {
        $now = CarbonImmutable::now();
        $staleBefore = $now->subSeconds((int) config('proxy-manager.check.stale_after_seconds'));
        $dueBefore = $now->subMinutes((int) config('proxy-manager.check.interval_minutes'));

        ProxyServer::query()
            ->where('status', ProxyStatus::Checking)
            ->whereNotNull('checking_started_at')
            ->where('checking_started_at', '<', $staleBefore)
            ->orderBy('id')
            ->chunkById(200, function ($proxies) use ($applyResult, $now): void {
                foreach ($proxies as $proxy) {
                    $startedAt = CarbonImmutable::parse($proxy->checking_started_at);

                    $applyResult->execute(
                        $proxy,
                        new ProxyCheckResult(
                            ProxyStatus::Offline,
                            $startedAt,
                            $now,
                            null,
                            null,
                            ProxyCheckErrorCode::StaleCheck,
                            'Proxy check became stale.',
                        ),
                        ProxyCheckSource::Auto,
                    );
                }
            });

        ProxyServer::query()
            ->where(function ($query) use ($dueBefore): void {
                $query->whereNull('last_checked_at')
                    ->orWhere('last_checked_at', '<=', $dueBefore);
            })
            ->where(function ($query) use ($staleBefore): void {
                $query->where('status', '!=', ProxyStatus::Checking)
                    ->orWhereNull('checking_started_at')
                    ->orWhere('checking_started_at', '<', $staleBefore);
            })
            ->orderBy('id')
            ->chunkById(200, function ($proxies) use ($scheduleProxyCheck): void {
                foreach ($proxies as $proxy) {
                    $scheduleProxyCheck->execute($proxy, ProxyCheckSource::Auto);
                }
            });
    }
}
```

```php
use Illuminate\Http\Resources\Json\AnonymousResourceCollection;
use Illuminate\Http\Response;

class ProxyController extends Controller
{
    public function index(IndexProxyRequest $request): AnonymousResourceCollection
    {
        $validated = $request->validated();
        $perPage = (int) ($validated['per_page'] ?? 20);
        $sort = $validated['sort'] ?? 'created_at';
        $direction = $validated['direction'] ?? 'desc';

        $query = ProxyServer::query();

        if (filled($validated['search'] ?? null)) {
            $search = $validated['search'];
            $query->where(function ($query) use ($search): void {
                $query->where('name', 'like', "%{$search}%")
                    ->orWhere('host', 'like', "%{$search}%")
                    ->orWhere('username', 'like', "%{$search}%");
            });
        }

        if (filled($validated['status'] ?? null)) {
            $query->where('status', $validated['status']);
        }

        if (filled($validated['scheme'] ?? null)) {
            $query->where('scheme', $validated['scheme']);
        }

        return ProxyResource::collection($query->orderBy($sort, $direction)->paginate($perPage));
    }

    public function store(StoreProxyRequest $request, CreateProxyAction $createProxy): JsonResponse|ProxyResource
    {
        try {
            return (new ProxyResource($createProxy->execute($request->validated())))
                ->response()
                ->setStatusCode(Response::HTTP_CREATED);
        } catch (DuplicateProxyException) {
            return $this->duplicateResponse();
        }
    }

    public function show(ProxyServer $proxy): ProxyResource
    {
        return new ProxyResource($proxy);
    }

    public function update(UpdateProxyRequest $request, ProxyServer $proxy, UpdateProxyAction $updateProxy): JsonResponse|ProxyResource
    {
        try {
            return new ProxyResource($updateProxy->execute($proxy, $request->validated()));
        } catch (DuplicateProxyException) {
            return $this->duplicateResponse();
        }
    }

    public function destroy(ProxyServer $proxy, DeleteProxyAction $deleteProxy): Response
    {
        $deleteProxy->execute($proxy);

        return response()->noContent();
    }

    public function check(ProxyServer $proxy, ScheduleProxyCheckAction $scheduleProxyCheck): JsonResponse
    {
        $scheduleProxyCheck->execute($proxy, ProxyCheckSource::Manual);

        return response()->json([
            'data' => [
                'id' => $proxy->id,
                'status' => 'checking',
                'queued' => true,
            ],
        ], Response::HTTP_ACCEPTED);
    }

    public function checkAll(ScheduleAllProxyChecksAction $scheduleAllProxyChecks): JsonResponse
    {
        $count = $scheduleAllProxyChecks->execute();

        return response()->json([
            'data' => [
                'queued' => true,
                'candidate_count' => $count,
            ],
        ], Response::HTTP_ACCEPTED);
    }

    private function duplicateResponse(): JsonResponse
    {
        return response()->json([
            'message' => 'Proxy already exists.',
            'errors' => [
                'host' => ['A proxy with the same scheme, host, port and username already exists.'],
            ],
        ], Response::HTTP_CONFLICT);
```

```ts
export class ApiError extends Error {

    message: string,
    public readonly status: number,
    public readonly errors: Record<string, string[]> = {},
  ) {
    super(message);
  }
}

const baseUrl = import.meta.env.VITE_API_BASE_URL ?? '/api/v1';

export async function requestJson<T>(path: string, options: RequestInit = {}): Promise<T> {
  const headers = new Headers(options.headers);
  headers.set('Accept', 'application/json');

  if (options.body && !headers.has('Content-Type')) {
    headers.set('Content-Type', 'application/json');
  }

  const response = await fetch(`${baseUrl}${path}`, {
    ...options,
    headers,
  });

  if (response.status === 204) {
    return undefined as T;
  }

  const payload = await response.json().catch(() => ({}));

  if (!response.ok) {
    throw new ApiError(payload.message ?? 'Request failed.', response.status, payload.errors ?? {});
  }

  return payload as T;
}
```

```ts
function queryString(params: Partial<ProxyFilters>): string {
  const search = new URLSearchParams();
```

```ts
  const checkingIds = reactive(new Set<number>());
  let listController: AbortController | null = null;
  const timers = new Set<number>();

  const isChecking = computed(() => items.value.some((proxy) => proxy.status === 'checking'));

  async function load(): Promise<void> {
    listController?.abort();
    listController = new AbortController();
    loading.value = true;
    error.value = null;

    try {
      const response = await listProxies(filters, listController.signal);
      items.value = response.data;
      meta.value = response.meta;
    } catch (exception) {
      if ((exception as Error).name !== 'AbortError') {
        error.value = exception instanceof Error ? exception.message : 'Failed to load proxies.';
      }
    } finally {
      loading.value = false;
    }
  }

  async function create(payload: ProxyPayload): Promise<void> {
    await save(() => createProxy(payload));
  }

  async function update(id: number, payload: ProxyPayload): Promise<void> {
    await save(() => updateProxy(id, payload));
  }

  async function save(request: () => Promise<unknown>): Promise<void> {
    saving.value = true;
    error.value = null;
    fieldErrors.value = {};

    try {
      await request();
      await load();
    } catch (exception) {
      if (exception instanceof ApiError) {
        fieldErrors.value = exception.errors;
        error.value = exception.message;
      } else {
        error.value = 'Failed to save proxy.';
      }
      throw exception;
    } finally {
      saving.value = false;
    }
  }

  async function remove(id: number): Promise<void> {
    await deleteProxy(id);
    await load();
  }

  async function check(id: number): Promise<void> {
    checkingIds.add(id);
    await checkProxy(id);
    scheduleFocusedPolling(id);
    await load();
  }

  async function checkAll(): Promise<void> {
    await checkAllProxies();
    await load();
  }

  async function openHistory(id: number): Promise<void> {
    const response = await listProxyChecks(id);
    checks.value = response.data;
  }

  function scheduleFocusedPolling(id: number): void {
    let elapsed = 0;

    const tick = window.setInterval(async () => {
      elapsed += 3000;
      await load();

      const proxy = items.value.find((item) => item.id === id);

      if (!proxy || proxy.status !== 'checking' || elapsed >= 30000) {
        checkingIds.delete(id);
        window.clearInterval(tick);
        timers.delete(tick);
      }
    }, 3000);

    const first = window.setTimeout(() => {
      void load();
      timers.delete(first);
    }, 1000);

    timers.add(tick);
    timers.add(first);
  }

  const globalTimer = window.setInterval(() => {
    void load();
  }, 30000);
  timers.add(globalTimer);

  onBeforeUnmount(() => {
    listController?.abort();
    timers.forEach((timer) => window.clearInterval(timer));
    timers.clear();
  });

  return {
    items,
    meta,
    checks,
    filters,
    loading,
    saving,
    error,
    fieldErrors,
    checkingIds,
    isChecking,
    load,
    create,
    update,
    remove,
    check,
    checkAll,
    openHistory,
  };
}
```

```bash
cp .env.example .env
docker compose build

docker compose run --rm php-fpm php artisan key:generate
docker compose run --rm php-fpm php artisan migrate
docker compose up -d
```

## Queue And Scheduler

The `queue` service runs:

```bash
php artisan queue:work database --queue=proxy-checks,default --sleep=1 --tries=1 --timeout=30
```

The `scheduler` service runs:

```bash
php artisan schedule:work
```

Production still needs one system cron if scheduler is not run as a long-lived container:

```bash
* * * * * cd /path/to/app && php artisan schedule:run >> /dev/null 2>&1
```

## Status Rules

New and manually refreshed proxies are queued for asynchronous checking. The queue job calls the configured check URL through the proxy and writes `online` or `offline` results to `proxy_checks`. Stale `checking` rows are marked `offline` by the scheduled dispatcher.
